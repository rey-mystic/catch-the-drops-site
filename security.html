Anti-Scam Rules

Follow these every time. No exceptions.

1) Verify official domains — Use bookmarks. Avoid look-alikes.
2) Never click DM links — Airdrop hunters are phishing targets.
3) Use burner wallets — Main wallet stays cold.
4) Revoke approvals regularly — Don’t leave unlimited approvals.
5) Check signatures — If it asks for “setApprovalForAll”, run.
